Adobe DreamWeaver 

O que é e para que serve o dreamWeaver? 

É um aplicativo de desenvolvimento web que combina uma interface visual com um editor de código, tornando-o uma ferramenta poderosa para a criação, gestão e modificação de websites e aplicações web.

Sites dinâmicos; Páginas de comércio eletrônico; Blogs; Boletins informativos; Portfólios, entre outros.

Como trabalhamos ou fazemos. 

Temos duas opções de usarmos o DreamWeaver, podemos criar os conteúdos dos sites incorporando os elementos por itens nos Menus (comandos), ou criarmos o site programando (TAGs) com os códigos em HTML, CSS e outras linguagens de programação Web.
